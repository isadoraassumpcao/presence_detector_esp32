# Presence Detector - Electronic Project with ESP32, Ultrasonic Sensor, Buzzer
## Components I used :
- 1 Ultrasonic Distance Sensor
- 2 LED (1 red, 1 green)
- 1 Active Buzzer
- 2 Resistors 330 ohm each
- ESP 32 (I used ESP32 DEV KIT DOIT)
- Jumper wires
